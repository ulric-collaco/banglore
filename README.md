# BESCOM EV Intelligence Dashboard

3D EV charging planning dashboard for Bengaluru.

## What It Shows
- Live load monitor with animated traffic flow
- Infrastructure planner with recommended new charging sites
- Science mode with clustering, regression, and scoring visuals

## Run Locally
```bash
cd bescom-ev-dashboard
npm install
npm run dev
```

## Build
```bash
cd bescom-ev-dashboard
npm run build
```

## Tech
React + Vite + deck.gl + OpenStreetMap tiles
